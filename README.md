# Gargun Valley
Prototpye of a farming game

[Trello Board](https://trello.com/b/7xYE8xfg/gargun-valley)

#### Version 0.0.1

## Description
A farming game with shooting elements. 

## Controls

- **Move** [A], [S], [W], [D]
- **Aim** Mouse
- **Shoot** Left Mouse Click
- **Light Toogle** [R]
- **Pause** Esc
